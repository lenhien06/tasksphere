# TaskSphere Landing Page

Landing page giới thiệu dự án **TaskSphere** theo hướng rõ ràng, chuyên nghiệp và phù hợp cho demo sản phẩm.

🌐 Landing page: https://lenhien06.github.io/tasksphere

- [Backend](https://github.com/lenhien06/tasksphere-backend) - Java 21 · Spring Boot 3.5.5 · MySQL · Redis · WebSocket
- [Frontend](https://github.com/lenhien06/tasksphere-frontend) - Next.js 15 · React 18 · TypeScript · TanStack Query · Tailwind CSS

## Mục tiêu của trang

Trang này tập trung thể hiện:

- giá trị sản phẩm đối với doanh nghiệp
- các nhóm tính năng lớn của hệ thống
- công nghệ sử dụng ở backend và frontend
- các thuật toán và business rules nổi bật
- thông tin đội ngũ phát triển

Landing page được viết dưới dạng **static HTML/CSS**, phù hợp để demo nhanh, trình chiếu hoặc deploy lên GitHub Pages.

## Cấu trúc thư mục

- `index.html`
  Landing page chính của dự án.

- `team-photos/`
  Ảnh thành viên và ảnh nhóm dùng trong phần giới thiệu đội ngũ.

## Nội dung nổi bật trên landing page

- Giới thiệu tổng quan về TaskSphere
- Giá trị hệ thống mang lại cho doanh nghiệp
- Các nhóm tính năng lớn như workspace health, sprint planning, Kanban, reporting, AI planning, notifications
- Phần thuật toán áp dụng như AI scoring, capacity detection, burndown/velocity, dependency rules, burnout detector
- Giới thiệu đội ngũ với ảnh thật và vai trò ngắn gọn: `Leader`, `Member`

## Cách chỉnh sửa

Chỉ cần mở và sửa trực tiếp:

```html
index.html
```

Nếu cần thay ảnh thành viên:

1. cập nhật file ảnh trong thư mục `team-photos/`
2. sửa lại đường dẫn ảnh tương ứng trong `index.html`

## Deploy nhanh

Vì đây là static page, có thể deploy bằng nhiều cách:

- GitHub Pages
- Netlify
- Vercel
- hoặc mở trực tiếp `index.html` trong trình duyệt để demo offline
