Về dự án

TaskSphere ra đời
như thế nào?

TaskSphere được phát triển như một dự án thực chiến nhằm giải quyết bài toán quản lý công việc nhóm trong môi trường học tập và làm việc — nơi mà các công cụ hiện có hoặc quá phức tạp, hoặc thiếu tính năng cộng tác thời gian thực.

Chúng tôi xây dựng hệ thống với kiến trúc client–server tách biệt hoàn toàn, đặt trọng tâm vào khả năng mở rộng, hiệu suất và trải nghiệm người dùng mượt mà — kể cả khi nhiều thành viên cùng thao tác đồng thời.

Dự án áp dụng đầy đủ các kỹ thuật hiện đại: JWT authentication, Redis caching, WebSocket realtime, TanStack Query cho server state, và Tailwind CSS utility-first cho giao diện responsive.

Tính năng

Đầy đủ tính năng
cho nhóm làm việc

Từ quản lý task cơ bản đến cộng tác realtime nâng cao — TaskSphere cung cấp mọi công cụ cần thiết.

📋
Quản lý Task & Project

Tạo và quản lý nhiều dự án song song. Trong mỗi project có thể tạo task, gán thành viên phụ trách, đặt mức độ ưu tiên, deadline và theo dõi trạng thái theo luồng Kanban linh hoạt.

CRUD Project Kanban Board Priority
⚡
Realtime Collaboration

Mọi thao tác như tạo task, thay đổi trạng thái, hay comment đều được đồng bộ tức thì đến toàn bộ thành viên đang online qua WebSocket — không cần reload trang.

WebSocket Live Sync Online Status
🔐
Xác thực & Phân quyền

Hệ thống xác thực JWT với access token và refresh token. Phân quyền theo vai trò: Admin có toàn quyền quản lý, Member chỉ thao tác trong phạm vi được giao.

JWT Auth Role-based Refresh Token
🚀
Redis Caching

Session và dữ liệu truy cập thường xuyên được cache trên Redis, giúp giảm đáng kể số lần query database, tăng tốc độ phản hồi API và cải thiện trải nghiệm người dùng.

In-memory Cache Session Store
🔄
TanStack Query

Server state được quản lý thông minh bởi TanStack Query — tự động cache response, refetch khi stale, retry khi lỗi mạng và đồng bộ dữ liệu giữa nhiều component.

Smart Cache Auto Refetch Optimistic UI
📱
Responsive UI

Toàn bộ giao diện được xây dựng bằng Tailwind CSS với thiết kế mobile-first, đảm bảo trải nghiệm nhất quán trên mọi kích thước màn hình từ điện thoại đến desktop.

Mobile-first Tailwind CSS Dark Mode
